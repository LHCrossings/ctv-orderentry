Saving and retrieving application logs

Step-by-step log procedures for Automation (CIB), Scheduling, and Aligner, plus how to pull a Windows Event Viewer log when a module-level log isn't enough.

Procedure — saving application logs
CIB Automation (CIB)
  1. Click the "Message Logs" button in the toolbar at the top of the window.
  2. Click Save.

Saving logs dialog in Automation (CIB)

SCH Scheduling
  1. From the main menu, click the Tools section.
  2. Click Save Logs.

Tools menu and Saving Logs dialog in Scheduling

ALN Aligner
  1. Click the "Save Logs" button in the toolbar at the top of the window.
  2. Click Save.

Note: Aligner runs in the background as its own app. You can find it on the CIB taskbar.

Saving logs dialog in Aligner

Procedure — Windows Event Viewer
EVT Event Viewer (Application and System)

This is the procedure:

  1. Open Event Viewer.
  2. Select Windows Logs.
  3. Select Application and System.
  4. Select English as the display language.
  5. In the right-hand column, click "Save All Events As…"
  6. Grab the files from the save folder and send them to Etere support (upload to a OneDrive folder).

Note: This information must be retrieved from the CIB workstations.
