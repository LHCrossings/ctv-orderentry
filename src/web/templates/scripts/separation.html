Update Contract Line Separation

Sets customer, event, and order separation intervals (in minutes) on every line in an Etere contract.

Separation (minutes)
Customer
Sets the interval between customers when checking the product code.
Order
Sets the interval between each Order.
Event
Sets the interval time between events when applying rules.
Common values — Customer/Order/Event: 15/0/0 (standard) · 25/0/0 (30-min rule) · 10/0/0 (TCAA) · 0/0/0 (billboard)
